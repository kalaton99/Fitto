4.1 WHY INVALID MODEL NAMES CAUSED 500 ERRORS
----------------------------------------------

REQUEST FLOW:
1. Client sends message to /api/ai-coach
2. Server validates input and API key ✅
3. Server calls GLM API with model name ❌
4. GLM API returns 400 with error code 1211 (Model Not Found)
5. Fallback logic tries next model ❌ (all were invalid)
6. Server returns 500 to client

ERROR RESPONSE FROM GLM:
{
  "error": {
    "code": "1211",
    "message": "Model not found: glm-4-air"
  }
}
